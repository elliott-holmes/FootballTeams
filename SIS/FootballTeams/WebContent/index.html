This is the main screen for the football teams.

To find a team by name go to /FootballTeams/rs/Football/getTeam/{teamName} - Team names are case sensitive so just keep an eye on that (Cant give you a hyperlink as you need to include the team name at the end).

To get all the teams go to rs/Football/getTeams

To get all the teams sorted by Capacity go to rs/Football/getTeamsByCapacity

To add a new team, you can use the button below :-
or post a json string to rs/Football/addTeam with the fields :-

  • name
  • city
  • owner
  • stadiumCapacity - Must be a number
  • competition
  • numberOfPlayers - another number
  • dateOfCreation - Must be a date formatted string (go for yyyy-mm-ddT00:00:00Z)
